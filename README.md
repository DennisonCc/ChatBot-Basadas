# ChatBot SmartHR - Basadas en Conocimiento

Asistente virtual inteligente para la gestión de recursos humanos, integrado con SmartHR.

## Características

*   **Asistente IA:** Responde consultas sobre turnos, pausas, asistencia y nómina.
*   **Interfaz Moderna:** Dashboard estilo Cyberpunk/Profesional.
*   **Demo Next.js:** Integración frontend completa.

## Instalación

1.  Clonar repositorio: `git clone https://github.com/DennisonCc/ChatBot-Basadas.git`
2.  Backend: `pip install -r requirements.txt` -> `python main.py`
3.  Frontend: `cd demo-next` -> `npm install` -> `npm run dev`
